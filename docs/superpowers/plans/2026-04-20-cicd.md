# CI/CD Pipeline Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Set up GitHub Actions CI/CD with test, lint, Nix build, and OCI container publishing to ghcr.io.

**Architecture:** Two workflow files — `ci.yml` runs test/lint/build on every push and PR, `container.yml` builds and pushes OCI images on main push and version tags. Ruff added as linter. Frontend stub unblocks `nix-build`.

**Tech Stack:** GitHub Actions, Nix, DeterminateSystems actions, Ruff, pytest, dockerTools

---

## File Structure

| File | Responsibility |
|------|----------------|
| `frontend.nix` | Stub Nix derivation producing placeholder frontend assets |
| `pyproject.toml` | Add ruff dev dependency and `[tool.ruff]` configuration |
| `shell.nix` | Add ruff to dev shell buildInputs |
| `.github/workflows/ci.yml` | CI workflow: test, lint, build jobs |
| `.github/workflows/container.yml` | Container workflow: build and push OCI image |

### Task 2: CI Workflow

**Files:**
- Create: `.github/workflows/ci.yml`

- [ ] **Step 1: Create the directory**

```bash
mkdir -p .github/workflows
```

- [ ] **Step 2: Create `.github/workflows/ci.yml`**

```yaml
name: CI

on:
  push:
    branches: [main]
  pull_request:
    branches: [main]

jobs:
  test:
    name: Tests
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: DeterminateSystems/nix-installer-action@main
      - uses: DeterminateSystems/magic-nix-cache-action@main
      - name: Run tests
        run: nix-shell --run "python -m pytest tests/ -v"

  lint:
    name: Lint
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: DeterminateSystems/nix-installer-action@main
      - uses: DeterminateSystems/magic-nix-cache-action@main
      - name: Ruff check
        run: nix-shell --run "ruff check backend/ tests/"
      - name: Ruff format check
        run: nix-shell --run "ruff format --check backend/ tests/"

  build:
    name: Nix Build
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4
      - uses: DeterminateSystems/nix-installer-action@main
      - uses: DeterminateSystems/magic-nix-cache-action@main
      - name: Build package
        run: nix-build default.nix
```

- [ ] **Step 3: Commit**

```bash

git commit -m "feat: add CI workflow (test, lint, build)"
```

### Task 3: Container Workflow

**Files:**
- Create: `.github/workflows/container.yml`

- [ ] **Step 1: Create `.github/workflows/container.yml`**

```yaml
name: Container

on:
  push:
    branches: [main]
    tags: ["v*"]

permissions:
  contents: read
  packages: write

env:
  REGISTRY: ghcr.io
  IMAGE_NAME: ${{ github.repository }}

jobs:
  build-and-push:
    name: Build and Push
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - uses: DeterminateSystems/nix-installer-action@main
      - uses: DeterminateSystems/magic-nix-cache-action@main

      - name: Build OCI image
        run: nix-build oci.nix

      - name: Load image into Docker
        run: docker load < result

      - name: Log in to GitHub Container Registry
        uses: docker/login-action@v3
        with:
          registry: ${{ env.REGISTRY }}
          username: ${{ github.actor }}
          password: ${{ secrets.GITHUB_TOKEN }}

      - name: Tag and push (latest)
        if: github.ref == 'refs/heads/main'
        run: |
          docker tag skynetcontrol:latest ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest
          docker push ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest

      - name: Tag and push (version)
        if: startsWith(github.ref, 'refs/tags/v')
        run: |
          VERSION=${GITHUB_REF#refs/tags/v}
          docker tag skynetcontrol:latest ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:${VERSION}
          docker tag skynetcontrol:latest ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest
          docker push ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:${VERSION}
          docker push ${{ env.REGISTRY }}/${{ env.IMAGE_NAME }}:latest
```

- [ ] **Step 2: Commit**

```bash
git add .github/workflows/container.yml
git commit -m "feat: add container workflow (build and push to ghcr.io)"
```
