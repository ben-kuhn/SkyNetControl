# Multi-Net Support Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make "net" a first-class container that owns all net-scoped data and routes; ship multi-tenant support with per-net memberships, slug-in-URL routing, and a one-shot migration that lands existing data in a "Default Net".

**Architecture:** Add three new tables (`nets`, `net_memberships`, `net_config`). Add `net_id` to every per-net table. Replace global `User.role` with global `is_admin` + per-net `net_memberships.role`. Mount net-scoped routes under `/api/nets/{slug}/...`. Frontend routes mirror at `/nets/:slug/...`. A single Alembic migration backfills everything into a Default Net.

**Tech Stack:** FastAPI, SQLAlchemy 2.x, Alembic, React + TypeScript, React Router v6, Vite.

## Global Constraints

- **Spec:** `docs/superpowers/specs/2026-06-22-multi-net-design.md` — every task implements that spec.
- **Linting:** `ruff check` clean (line-length 120, select `E`,`F`).
- **Toolchain:** All Python commands run via `.venv/bin/...` or `nix-shell --run "..."`. Frontend via `cd frontend && nix-shell -p nodejs_22 --run "npm <cmd>"`. Never pip-install into host Python.
- **Commits:** Conventional Commits with `(multi-net)` scope where it clarifies — e.g. `feat(multi-net): add nets table`, `refactor(multi-net): route schedule under net slug`.
- **DB:** SQLite for dev. The migration must run cleanly against the developer's current dev DB (which has one net's worth of data) and against an empty DB.
- **Migration:** One Alembic revision. Downgrade body raises `NotImplementedError` (pre-alpha; no rollback).
- **Slug grammar:** 1-64 chars, `[a-z0-9-]`, must start and end alphanumeric, no consecutive hyphens. Enforced server-side on create/rename.
- **No public-net leakage:** Net-scoped GET endpoints must verify net membership OR (`is_public` AND endpoint is in the public-read allow-list).
- **Test discipline:** Each task ends green: `.venv/bin/pytest -q` passes, `ruff check` passes. Add tests *with* the code they cover.
- **Dev server:** Each task leaves `./run-dev.sh` runnable.

```python
# backend/modules/nets/models.py
import enum
from datetime import datetime, timezone

from sqlalchemy import (
    Boolean,
    DateTime,
    Enum,
    ForeignKey,
    Integer,
    String,
    Text,
)
from sqlalchemy.orm import Mapped, mapped_column, relationship

from backend.db.base import Base


class NetRole(str, enum.Enum):
    NET_CONTROL = "net_control"
    VIEWER = "viewer"


class Net(Base):
    __tablename__ = "nets"

    id: Mapped[int] = mapped_column(Integer, primary_key=True, autoincrement=True)
    slug: Mapped[str] = mapped_column(String(64), unique=True, nullable=False)
    name: Mapped[str] = mapped_column(String(255), nullable=False)
    is_public: Mapped[bool] = mapped_column(Boolean, nullable=False, default=True, server_default="1")
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True),
        nullable=False,
        default=lambda: datetime.now(timezone.utc),
    )

    memberships: Mapped[list["NetMembership"]] = relationship(
        back_populates="net", cascade="all, delete-orphan"
    )


class NetMembership(Base):
    __tablename__ = "net_memberships"

    user_callsign: Mapped[str] = mapped_column(
        String(20), ForeignKey("users.callsign"), primary_key=True
    )
    net_id: Mapped[int] = mapped_column(
        Integer, ForeignKey("nets.id"), primary_key=True
    )
    role: Mapped[NetRole] = mapped_column(Enum(NetRole), nullable=False)
    created_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True),
        nullable=False,
        default=lambda: datetime.now(timezone.utc),
    )

    net: Mapped["Net"] = relationship(back_populates="memberships")


class NetConfig(Base):
    __tablename__ = "net_config"

    net_id: Mapped[int] = mapped_column(
        Integer, ForeignKey("nets.id"), primary_key=True
    )
    key: Mapped[str] = mapped_column(String(255), primary_key=True)
    value: Mapped[str] = mapped_column(Text, nullable=False)
    updated_at: Mapped[datetime] = mapped_column(
        DateTime(timezone=True),
        nullable=False,
        default=lambda: datetime.now(timezone.utc),
        onupdate=lambda: datetime.now(timezone.utc),
    )
```

## Task 3: Big migration — backfill into Default Net

The single one-shot revision that creates the nets row, adds `net_id` columns everywhere, backfills, rewrites uniqueness constraints, and finally drops `User.role`.

**Files:**
- Create: `alembic/versions/<rev>_multi_net.py`
- Test: `tests/test_migration_multi_net.py`

**Interfaces produced (post-migration database shape):**
- `nets` has exactly one row with `id=1`.
- Every existing `net_seasons`, `roster_templates`, `reminder_templates`, `activities`, `activity_tags`, `members` row carries `net_id=1`.
- `members` PK is `(net_id, callsign)`.
- `roster_templates.name`, `reminder_templates.name`, `activity_tags.name` are unique within `net_id`.
- `users.role` removed; `is_admin` populated.
- Every non-admin, non-pending, non-deleted user has a `net_memberships(net_id=1, role)` row matching their old `role`.
- `personal_access_tokens.net_id` populated: scoped tokens → 1, admin-only tokens → NULL.
- `audit_log.net_id` populated for best-effort known-net events; otherwise NULL.
- Per-net `app_config` keys (`net_address`, `default_net_control`, `pat_mailbox_path`, `scanner.*`, `delivery.*`, `callbook.*`) copied into `net_config(net_id=1, …)`, then deleted from `app_config`.

- [ ] **Step 2: Write the upgrade body**

Write the upgrade in 10 helper sections (one per migration step in the spec). Use `op.batch_alter_table` for SQLite-compatible column adds, PK changes, and unique-constraint replacements. Pseudocode at high level:

```python
def upgrade() -> None:
    # 1. New tables
    op.create_table(
        "nets",
        sa.Column("id", sa.Integer, primary_key=True, autoincrement=True),
        sa.Column("slug", sa.String(64), nullable=False, unique=True),
        sa.Column("name", sa.String(255), nullable=False),
        sa.Column("is_public", sa.Boolean, nullable=False, server_default="1"),
        sa.Column("created_at", sa.DateTime(timezone=True), nullable=False),
    )
    op.create_table(
        "net_memberships",
        sa.Column("user_callsign", sa.String(20), sa.ForeignKey("users.callsign"), primary_key=True),
        sa.Column("net_id", sa.Integer, sa.ForeignKey("nets.id"), primary_key=True),
        sa.Column("role", sa.String(20), nullable=False),
        sa.Column("created_at", sa.DateTime(timezone=True), nullable=False),
    )
    op.create_table(
        "net_config",
        sa.Column("net_id", sa.Integer, sa.ForeignKey("nets.id"), primary_key=True),
        sa.Column("key", sa.String(255), primary_key=True),
        sa.Column("value", sa.Text, nullable=False),
        sa.Column("updated_at", sa.DateTime(timezone=True), nullable=False),
    )

    # 2. Insert default net
    bind = op.get_bind()
    net_address = bind.execute(sa.text(
        "SELECT value FROM app_config WHERE key = 'net_address'"
    )).scalar()
    name = net_address or "Default Net"
    slug = _slugify(name) or "default"
    now = datetime.now(timezone.utc).isoformat()
    bind.execute(sa.text(
        "INSERT INTO nets (id, slug, name, is_public, created_at) "
        "VALUES (1, :slug, :name, 1, :now)"
    ), {"slug": slug, "name": name, "now": now})

    # 3. Add net_id columns and backfill
    for tbl in ("net_seasons", "roster_templates", "reminder_templates",
                "activities", "activity_tags"):
        with op.batch_alter_table(tbl) as batch:
            batch.add_column(sa.Column("net_id", sa.Integer, sa.ForeignKey("nets.id")))
        op.execute(f"UPDATE {tbl} SET net_id = 1")
        with op.batch_alter_table(tbl) as batch:
            batch.alter_column("net_id", nullable=False)

    # 4. Replace template / tag UNIQUE(name) with UNIQUE(net_id, name)
    for tbl, cname in (
        ("roster_templates", "uq_roster_templates_name"),
        ("reminder_templates", "uq_reminder_templates_name"),
        ("activity_tags", "uq_activity_tags_name"),
    ):
        with op.batch_alter_table(tbl) as batch:
            batch.drop_constraint(cname, type_="unique")  # name may vary; check actual constraint name
            batch.create_unique_constraint(f"uq_{tbl}_net_name", ["net_id", "name"])

    # 5. members PK → (net_id, callsign)
    with op.batch_alter_table("members") as batch:
        batch.add_column(sa.Column("net_id", sa.Integer, sa.ForeignKey("nets.id")))
    op.execute("UPDATE members SET net_id = 1")
    with op.batch_alter_table("members", recreate="always") as batch:
        batch.alter_column("net_id", nullable=False)
        batch.drop_constraint("pk_members", type_="primary")
        batch.create_primary_key("pk_members", ["net_id", "callsign"])

    # 6. Backfill net_memberships from users.role
    op.execute("""
        INSERT INTO net_memberships (user_callsign, net_id, role, created_at)
        SELECT callsign, 1, role, :now
        FROM users
        WHERE role IN ('net_control', 'viewer')
    """.replace(":now", f"'{now}'"))

    # 7. Drop users.role
    with op.batch_alter_table("users") as batch:
        batch.drop_column("role")

    # 8. personal_access_tokens.net_id
    with op.batch_alter_table("personal_access_tokens") as batch:
        batch.add_column(sa.Column("net_id", sa.Integer, sa.ForeignKey("nets.id"), nullable=True))
    # Admin-only scopes (users:*, config:*, nets:*) → keep NULL
    # Everything else with a scope list → net 1
    op.execute("""
        UPDATE personal_access_tokens
        SET net_id = 1
        WHERE NOT (
            scopes LIKE '%users:%' OR scopes LIKE '%config:%' OR scopes LIKE '%nets:%'
        )
    """)

    # 9. audit_log.net_id
    with op.batch_alter_table("audit_log") as batch:
        batch.add_column(sa.Column("net_id", sa.Integer, nullable=True))
    # Best-effort: events whose action prefix is per-net domain → 1
    op.execute("""
        UPDATE audit_log SET net_id = 1
        WHERE action LIKE 'schedule.%'
           OR action LIKE 'checkin%'
           OR action LIKE 'roster.%'
           OR action LIKE 'reminder.%'
           OR action LIKE 'activity.%'
    """)

    # 10. Move per-net app_config keys to net_config
    PER_NET_PREFIXES = (
        "net_address", "default_net_control", "pat_mailbox_path",
        "scanner.", "delivery.", "callbook.",
    )
    rows = bind.execute(sa.text("SELECT key, value, updated_at FROM app_config")).fetchall()
    for key, value, updated_at in rows:
        if any(key == p or (p.endswith(".") and key.startswith(p)) for p in PER_NET_PREFIXES):
            bind.execute(sa.text(
                "INSERT INTO net_config (net_id, key, value, updated_at) "
                "VALUES (1, :k, :v, :u)"
            ), {"k": key, "v": value, "u": updated_at})
            bind.execute(sa.text("DELETE FROM app_config WHERE key = :k"), {"k": key})


def downgrade() -> None:
    raise NotImplementedError("multi-net cutover is one-way (pre-alpha)")


def _slugify(s: str) -> str:
    s = s.lower()
    out = []
    last_hyphen = False
    for c in s:
        if c.isalnum():
            out.append(c)
            last_hyphen = False
        elif not last_hyphen:
            out.append("-")
            last_hyphen = True
    return "".join(out).strip("-")[:64]
```

```python
from dataclasses import dataclass
from fastapi import Path
from backend.modules.nets.models import Net, NetMembership, NetRole

_NET_ROLE_RANK = {NetRole.VIEWER: 1, NetRole.NET_CONTROL: 2}


@dataclass
class NetContext:
    user: User
    net: Net
    role: NetRole | None  # None if access is via is_admin


def require_admin(user: User = Depends(get_current_user)) -> User:
    if not user.is_admin:
        raise HTTPException(status_code=403, detail="Admin required")
    return user


def require_net_role(min_role: NetRole):
    min_rank = _NET_ROLE_RANK[min_role]

    def dep(
        net_slug: str = Path(..., alias="net_slug"),
        user: User = Depends(get_current_user),
        db: Session = Depends(get_db_session),
    ) -> NetContext:
        net = db.query(Net).filter(Net.slug == net_slug).one_or_none()
        if net is None:
            raise HTTPException(status_code=404, detail="Net not found")
        if user.is_admin:
            return NetContext(user=user, net=net, role=None)
        m = db.get(NetMembership, (user.callsign, net.id))
        if m is None or _NET_ROLE_RANK[m.role] < min_rank:
            raise HTTPException(status_code=403, detail="Insufficient permissions for net")
        return NetContext(user=user, net=net, role=m.role)

    return dep
```

```tsx
// frontend/src/context/CurrentNetContext.tsx
import { createContext, useContext, useEffect, useState } from "react";
import { useParams } from "react-router-dom";
import { getNet } from "../api/nets";
import { useAuth } from "./AuthContext";

interface Ctx {
    slug: string;
    net: NetSummary | null;
    role: NetRole | "admin" | null;
    loading: boolean;
    error: string | null;
}

const CurrentNetContext = createContext<Ctx | undefined>(undefined);

export function CurrentNetProvider({ children }: { children: React.ReactNode }) {
    const { slug } = useParams<{ slug: string }>();
    const { user } = useAuth();
    const [net, setNet] = useState<NetSummary | null>(null);
    const [error, setError] = useState<string | null>(null);

    useEffect(() => {
        if (!slug) return;
        getNet(slug).then(setNet).catch(e => setError(String(e)));
        localStorage.setItem("lastNetSlug", slug);
    }, [slug]);

    const role: NetRole | "admin" | null =
        user?.isAdmin ? "admin" :
        user?.nets.find(n => n.slug === slug)?.role ?? null;

    return (
        <CurrentNetContext.Provider value={{
            slug: slug!, net, role, loading: !net && !error, error,
        }}>
            {children}
        </CurrentNetContext.Provider>
    );
}

export function useCurrentNet() {
    const c = useContext(CurrentNetContext);
    if (!c) throw new Error("useCurrentNet outside provider");
    return c;
}
```

## Notes for the implementer

- Tasks 6-11 are highly repetitive. After Task 6, an implementer should be able to do 7-11 in close to a single pass per module. Keep each commit focused per the conventional-commits guidance in CLAUDE.md.
- The migration in Task 3 is the highest-risk step. Take the backup (Task 3 step 6); if anything goes sideways, restore from `skynetcontrol.db.pre-multi-net` and rerun.
- When you hit a constraint name mismatch in batch_alter_table, the actual constraint name in SQLite is usually `sqlite_autoindex_<table>_<n>` for unique constraints. `op.batch_alter_table(..., recreate="always")` sidesteps that by rebuilding the table — use it whenever a constraint name is uncertain.
- Don't worry about downgrade. Pre-alpha.
- Template-default editing (Winlink-tailored seeds → kind-aware seeds) is explicitly deferred to a separate "live nets" feature branch.
